# Local Proxy Pool Subscription

This repository publishes a strict-filtered proxy subscription generated by `subs-check`.

## Subscription URLs

After GitHub Pages is enabled by the workflow, use:

```text
https://xiaorong666.github.io/local-proxy-pool-subscription/mihomo.yaml
https://xiaorong666.github.io/local-proxy-pool-subscription/all.yaml
https://xiaorong666.github.io/local-proxy-pool-subscription/base64.txt
```

Raw GitHub fallback:

```text
https://raw.githubusercontent.com/xiaorong666/local-proxy-pool-subscription/main/public/mihomo.yaml
https://raw.githubusercontent.com/xiaorong666/local-proxy-pool-subscription/main/public/all.yaml
https://raw.githubusercontent.com/xiaorong666/local-proxy-pool-subscription/main/public/base64.txt
```

## Current Filter Policy

- Strict timeout: 3000 ms
- Speed test enabled: `speed-test-url: random`
- Minimum speed: 512 KB/s
- Media check enabled
- Bad Cloudflare nodes dropped
- Protocols kept: `ss`, `vmess`, `vless`, `trojan`
- Historical success reuse disabled

## Run Manually

Open Actions, choose `Update proxy subscription`, then click `Run workflow`.

The workflow also runs every 6 hours.
